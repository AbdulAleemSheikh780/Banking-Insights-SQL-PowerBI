Customer Insights For Banking 🏦📊
An end-to-end data analysis project focusing on financial metrics, liquidity, and customer segmentation using SQL (PostgreSQL) and Power BI.

🚀 Project Overview
This project transforms raw banking transaction data (1,000+ records) into actionable business insights. The goal is to provide bank managers with a clear view of branch performance, currency distribution, and financial health.

🛠️ Tech Stack
Database: PostgreSQL (Advanced SQL Queries)

Visualization: Power BI

Data Source: MS Excel (Kaggle Dataset)

Mathematics: Financial Ratios & Statistical Aggregations

📈 Key Insights & Features
Liquidity Ratio: Calculated the bank's financial stability across branches.

Customer Segmentation: Used SQL CASE statements to categorize wealth tiers (Silver, Gold, Platinum).

Branch Performance: Ranked branches based on transaction volume and account balances.

Interactive Analysis: AI-powered Decomposition Tree for balance deep-dives.

📁 File Structure
01_Raw_Data: The original source data.

02_Banking_Queries.sql: Contains 23+ professional SQL scripts.

03_Project_Presentation.pptx: Slide-by-slide project walk-through.

04_Banking_Dashboard.pbix: The final Power BI interactive file.
